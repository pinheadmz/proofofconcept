Built with Handout: https://github.com/pinheadmz/handout
Handout is a nodejs package that runs a webserver and an authoritative nameserver. With one simple command, the install script generates DNSSEC keys and a self-signed SSL certificate. The install script prints out Handshake HNS records for your TLD. When launched, the built-in nameserver generates all the DNS, DNSSEC and TLSA records to serve a Handshake website securely.
